Interfaces

CSharpInterface generates a public partial interface declaration. It works analogously to CSharpClass but emits properties without setters or bodies.

var myInterface = new CSharpInterface(new CSharpIdentifier("MyApp", "IMyService"))
{
    Summary = "Provides access to my service.",
    Interfaces = {new CSharpIdentifier("MyApp", "IBaseService")},
    Properties =
    {
        new CSharpProperty(CSharpIdentifier.String, "Name") {Summary = "The display name."}
    }
};

Add identifiers to Interfaces to declare base interfaces. Properties added via Properties are emitted as get-only members on the interface; any setters or initializers configured on a CSharpProperty only take effect when the property is used on a CSharpClass.
